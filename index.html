Русский Информ

Платформа для написания текстовых игр с парсерным вводом

О проекте Что это такое?

Русский Информ (RInform) — это перевод стандартной библиотеки популярной на западе системы Inform 6.

Игры Готовые творения на Информе

Готовая игра на Информе — это скомпилированный платформонезависимый файл с расширением .z5, .z8 или .zblorb. Чтобы запустить игру, нужно открыть её при помощи одной из программ-интерпретаторов, на выбор:

  • Windows Frotz для Windows.
  • Gargoyle — кроссплатформенный, ориентирован на типографику.
  • Parchment — написан на HTML5, работает прямо в браузере.
  • Zoom — для OS X и *nix систем.

Библиотека

Для разработки можно скачать рекомендуемую сборку (обновлена 7 ноября 2017), которая включает в себя компилятор (под Windows), библиотеку, демонстрационные игры и имеющуюся документацию.

Самая свежая версия библиотеки всегда доступна на репозитарии на BitBucket: https://bitbucket.org/yandexx/rinform. Патчи и баг-репорты сильно приветствуются.
