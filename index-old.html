RInform

Русский информ

Добро пожаловать на временный вариант сайта платформы для написания парсерных текстовых игр на русском языке. RInform представляет собой перевод стандартной библиотеки популярной на западе системы Inform 6.

Игры на платформе RInform:

  • Синие стулья (2011) — перевод Blue Chairs (2004)
  • Субботнее утро (2010) — фанфик по фильмам Джорджа Ромеро (автор: Сетке)
  • Вечерня (2010) — перевод Vespers (2005)
  • Робото (2010) — игра, написанная за один день на конкурс «Вжж!»
  • Spider And Web (2009) — перевод одноимённой игры (1998)
  • Метаморфозы (2008) — перевод Metamorphoses (2000)
  • Все пути (2007) — перевод All Roads (2001)
  • Приключение (2004) — перевод Adventure (1976)

Для запуска игр рекомендуются следующие интерпретаторы:

  • Windows Frotz (также в zip).
  • Gargoyle — начиная с версии 2010.1 поддерживается русский язык.

В данный момент для разработки можно скачать рекомендуемую сборку [2011-03-10], которая включает в себя компилятор (под Windows), библиотеку и демонстрационные игры.

Организован репозиторий на GitHub: https://github.com/yandexx/RInform

Документация (на русском языке):

  • Введение в Информ — перевод Inform Beginner's Guide с учётом особенностей русской версии платформы (готова 1 игра из 3).

Документация (на английском языке):

  • The Inform Beginner's Guide — рекомендуется новичкам.
